🛡️ Sistema de Gerenciamento de RPG
Um sistema idealizado para auxiliar Mestres de RPG a organizarem suas campanhas, permitindo a manipulação de uma biblioteca vasta de entidades e o controle em tempo real de sessões de jogo.

📌 Sobre o Projeto
Este é um projeto divergente (fork) desenvolvido a partir de um trabalho em grupo realizado durante o 2º semestre do curso de Análise e Desenvolvimento de Sistemas (ADS).

Atualmente, o Gerenciador de RPG (ainda sem nome oficial) busca expandir seus horizontes técnicos através da implementação de uma Interface Gráfica (GUI) e um Banco de Dados, além do desenvolvimento de novas features que tornam a narração mais dinâmica.

🚀 Novidades e Evoluções
O projeto agora conta com módulos específicos para diferentes momentos do jogo:

Cena: Interface que contém os locais de investigação, os itens a serem encontrados e as Dificuldades de Teste (DTs) para cada interação.

Inventário e Biblioteca de Itens: Visualização detalhada durante o combate, exibindo atributos como dano, defesa e poderes das entidades.

Biblioteca de Lugares: Repositório dedicado à descrição dos locais que aparecem nas cenas de investigação.

Sessão: Interface centralizada para visualização do roteiro do episódio e gerenciamento dos modos de jogo (Alternância entre Modo Combate e Modo Cena).
